## Contributing

### Ensuring Your GitHub Contributions Are Recorded

For your contributions to appear on your GitHub profile's contribution graph (green squares), ensure the following:

1. **Verify your commit email is linked to your GitHub account:**
   - Go to [GitHub Email Settings](https://github.com/settings/emails)
   - Add and verify the email address you use for commits
   - Check your current Git email: `git config user.email`
   - If needed, update it: `git config --global user.email "your-github-email@example.com"`

2. **Commits must be on the default branch or merged:**
   - Contributions on feature branches only count once merged to the default branch (main/master)
   - Make sure to merge your pull requests to see contributions reflected

3. **Repository requirements:**
   - The repository must not be a fork, OR
   - If it is a fork, contributions only count when merged to the parent repository

4. **Verify your setup:**
   ```bash
   # Check your Git configuration
   git config --global user.name
   git config --global user.email
   ```
